About the FPI

The Federal Program Inventory aggregates important information about the programs of the U.S. Government. This website builds on lessons learned during the FPI Exploratory Pilot, launched in 2020, by significatly increasing the number of programs, amount of spending, and information included.

Learn about the data

This pilot website uses publicly-available, agency-provided data. This data is sourced from SAM.gov and USASpending.gov. All data used on this website is available to download.
