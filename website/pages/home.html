The U.S. Government administers more than 2,300 federal financial assistance programs.

Search across categories

This pilot website allows visitors to explore federal financial assistance programs across categories, agencies, eligible applicants, and more.

Explore programs using advanced search

History of the pilot

This pilot website builds on the initial Federal Program Inventory, launched in 2020. This original exploratory pilot provided significant insights on how to display and explore the programs of the federal government.

Learn more about the exploratory pilot

Learn about the data

This pilot website uses publicly-available, agency-provided data. This data is sourced from SAM.gov and USASpending.gov to provide a holistic view of federal financial assistance programs. All data used on this website is available to download.
