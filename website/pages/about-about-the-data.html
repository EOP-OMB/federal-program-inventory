Overview

Pilot A leverages publicly accessible assistance listings (AL) data from SAM.gov and financial data from USAspending.gov, linked to assistance listings through AL number. The functions and subfunctions used to categorize the programs can be found in the Annual Publication of Assistance Listings Functional Index. Agencies submit data for their programs directly to these sources.

This website, including all underlying data, is open source and accessible at: [GITHUB LINK].

Data updates

This website and the underlying data are not continuously updated.The data used to build this website was last pulled on the following dates:

  • SAM.gov assistance listings: October 10, 2023
  • SAM.gov assistance listing categories & sub-categories: December 2022
  • USASpending.gov transaction data: August 8, 2023
